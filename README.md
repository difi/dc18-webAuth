#Id-porten redesign

### How to build

Prerequisites
- NPM
- Ruby
- SASS

1. `npm install`
2. `npm build`


### How to develop

1. `npm start`
This will automatically build the project in the `dev` folder as you edit them.


### Designing Webauthn for MinId
